# Tank 🐠

Hand-controlled interactive aquarium powered by computer vision.

## ✨ Features
- Real-time hand gesture control via MediaPipe
- Intelligent fish AI with dynamic behaviors
- ASCII art aesthetic with particle effects
- Mobile-responsive design

## 🎮 Interactions
- **Pinch** - Drop food
- **Open Hand** - Fish follow
- **Fist** - Scatter fish
